Frequently asked questions
Is Acronis True Image 2018 mirror imaging software?

Yes, but it also enables you to do so much more than traditional imaging software.

Not only can you create a full mirror image of your disk, Acronis True Image 2018 allows you to back up individual files, folders and partitions, clone a system disk while it is in use (no rebooting needed), and you can archive files to the cloud to open space on your hard drive. It will also back up your Facebook and Instagram data, as well as an unlimited number of mobile devices.

Does Acronis True Image 2018 do more than back up and recover data?

Absolutely. Acronis True Image 2018 also integrates wide variety of other features, including an automated WinPE boot media builder, an award-winning anti-ransomware solution, cloud storage, and blockchain certification of files. You can even use it to convert full image backups to a virtual hard disk format so you can run it as a virtual machine.

All of this makes Acronis True Image 2018 the most powerful personal backup solution on the market.

Is Acronis True Image 2018 available for Mac?
Yes. Acronis True Image 2018 supports Windows, macOS, Android, and iOS devices. It also is compatible with several network attached storage (NAS) devices, providing greater flexibility of local storage.
Is Acronis True Image 2018 only an online backup solution?

No. Acronis is a big proponent of the 3-2-1 backup rule, so Acronis True Image 2018 is designed to facilitate backups that are stored locally (on external hard drives, network drives and NAS devices) and in the cloud – although cloud storage and features require subscriptions to either the Advanced or Premium versions of Acronis True Image 2018.

With potent scheduling capabilities and flexible backups (including incremental and differential backups), Acronis True Image 2018 makes it easy to create both local and cloud-based backups.

How is Acronis True Image 2018’s anti-ransomware defence unique?

Any full image backup can help you recover your system. Criminals know this, so they are creating ransomware that targets backup files, software, and agents – robbing you of your recovery.

Acronis True Image 2018 is the only personal backup solution available that integrates an AI-based anti-ransomware technology designed to stop ransomware attacks in real-time – including a self-defence mechanism to ensure the program and the backups it creates are reliable and safe.
