Страница 15-летия Акрониса

23 июля 2018
  • acronis
  • work

Сайт похож на презентацию, поэтому для верстки использовался fullpage.js. Чтобы контент менялся горизонтально при скролле, пришлось купить плагин Scroll Horizontally.

Из-за встроенных видео с ютуба страница грузилась очень долго, поэтому я сделала lazy load. Для большей скорости подключила lazy load и в fullpage.js.

Форма на сайте отправляет данные в гугл форму. Для этого нужно создать гугл форму с такими же полями, и скопировать атрибуты name у полей и action у формы с гугл формы на свою. Наиболее понятный туториал нашла на сайте webjeda.com.
